Automatic Detection Of Genetic Diseases in Children Using Pupilometry

This project aims to detect genetic eye diseases in children using pupillometry, a technique that measures how the pupil reacts to light. By shining different colors of light into the eyes and recording the pupil's response, we can identify signs of Inherited Retinal Diseases (IRDs). We used machine learning (SVM) to analyze the data and help doctors make quick and accurate diagnoses.

Tech: Python, Machine Learning.
